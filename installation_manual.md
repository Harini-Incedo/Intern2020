## Install Angular: 

 [https://tecadmin.net/install-angular-on-macos/](https://tecadmin.net/install-angular-on-macos/)

``` $brew update ```

```$brew install node```

```$node -v```

```$npm install -g @angular/cli@9```   

```$ng --version```


If Permission denied - then try using **sudo** on Mac. Similarly root user for windows
